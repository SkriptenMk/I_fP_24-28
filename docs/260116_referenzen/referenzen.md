# Referenzen in Markdown

Damit in Texten, die im Markdown Format erstellt werden, Literaturzitate
eingefügt werden können, braucht es neben dem Markdown Dokument mit dem
eigentlichen Text, drei weitere Dateien:
eine Datei mit allen Formatierungsinformationen (`.yaml`), eine Datei
mit den bibliographischen Angaben (`.bib`) sowie eine Datei mit den
Formatierungsangaben für die Literaturdarstellung (`.csl`).
Von diesen drei Dateien muss lediglich die Datei mit den
Formatierungsinformationen manuell erstellt werden. Die Bibliographie
Datei wird von Zotero erstellt und die Datei für die Darstellung der
bibliographischen Angaben im Text kann aus dem [Zotero Style
Repository](https://www.zotero.org/styles) heruntergeladen werden.
