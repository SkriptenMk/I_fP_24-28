# Was ist eine Künstliche Intelligenz?

Sie alle kennen heute **generative K.I.**-Tools.
Wir werden zuerst K.I. anschauen, die keinen Inhalt generieren,
sondern Entscheidungen fällen. Dies ist zu Beginn einfacher nachzuvollziehen,
was ein Modell ist.
